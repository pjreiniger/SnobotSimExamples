# SnobotSimExamples
This provides examples on how to you can add [SnobotSim](https://github.com/pjreiniger/SnobotSim) support to your FRC project

Includes
* Simple Java Example
* Java example, overriding supported I2C and SPI devices
* C++ Project which can be simulated using the Java display provided by SnobotSim
* C++ Project that can be run using GradleRio's `gradlew simulateDesktopProgramReleaseExecutable` native debugging

Please leave all issues on the [SnobotSim](https://github.com/pjreiniger/SnobotSim/issues) repository
